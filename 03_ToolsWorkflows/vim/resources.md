# Useful vim resources

* Run `vimtutor` in your shell
* Cheat sheets (just search for *vim cheat sheet* online)
* [VIM Adventures](https://vim-adventures.com/) (vim-based adventure game)
* The built-in help through `:help topic`
* [Intermediate vim](https://dn.ht/intermediate-vim/)
* [Vimcasts](https://vimcasts.org) (screencasts)
* [VimAwesome](https://vimawesome.com/) (plugins)
